# odin-calculator
This is a beginner javascript project made via the Odin Project html and css course.
This exercise was the occasion to practice basic JS, especially project file organization, objects, event handling, functions, safeguard in function with control flow management and JSDoc.

## Elements studied through this project :
- problem solving method
- DOM manipulation
- event handling
- functions
- control flow
- global & local scope
- objects
- debugger
- JSDoc

## Many mistakes made through this project, this serves as a reminder on element to consider for future project :
- non atomic git commit were made
- functions call stack were hard to retrace sometimes making debugging difficult. Simplification and clear separation of concern in functions and file is required
